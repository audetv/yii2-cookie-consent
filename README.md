# Yii2 Cookie Consent Widget

Universal cookie consent notification widget for Yii2 framework.

## Features
- Easy installation via Composer
- Customizable text and button labels
- Configurable legal document links (Privacy Policy, Terms of Use)
- Light/Dark themes + custom CSS support
- Fixed position (bottom/top)
- I18n ready (Russian/English)

## Quick Start

\`\`\`php
echo \audetv\cookieconsent\CookieConsentWidget::widget([
    'theme' => 'dark',
    'privacyPolicyUrl' => '/privacy',
    'termsUrl' => '/terms',
]);
\`\`\`
